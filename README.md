# GAANet: Global-guided Asymmetric Attention Network for Audio-Visual Speech Separation

## Overview
This repository contains the official implementation of GAANet for audio-visual speech separation. The code is submitted for double-blind review.

![GAANet Architecture](assets/gaanet_architecture.png)

*Figure 1: Overall architecture of the proposed GAANet. The network consists of three main components: (1) Visual Encoder, (2) Global-guided Asymmetric Attention Module, and (3) Audio Separation Network.*
